Data Visualisation with WebGL

I recently worked on a data visualisation project for a networking company here in New Zealand. This involved creating a 3D representation of their network by overlaying network data onto a model of New Zealand. This was done using Three.js, a library for JavaScript that runs on top of WebGL.

A demo of the 3D point cloud of NZ can be seen here. (Press 'c' to watch the animation). If the link does not work (give it a few seconds...), try it in Chrome (the web browser with the most WebGL support).

Startup Weekend & Rover

Recently I attended an education themed Start Up Weekend in Wellington. This was a fantastic experience, in less than 48 hours our team of only 4 designed a business and a prototype for our product.

Our team's business idea was a Mobile Audience Response system, or a "MARS". Our application's name? Rover. Rover is a web app designed to increase student engagement, targetted at university lectures, but also suitable for high school classrooms. It works by allowing the teacher to ask the students questions via their laptops and/or mobile devices, and they can then see the response in real time.

Behind Rover is a PHP framework, along with SQL, HTML5, CSS3 and JavaScript code, which can be viewed here.

Visit the Start Up Weekend website to register for a Start Up Weekend near you.

NCSS & Wish List

A social media inspired web application. This is a website which allows users to log on, create a list of things they want for Christmas/Birthday/Anniversary etc, which their friends can then see.

This was created by a team of about 15 students, along with 5 teachers and tutors at the NCSS Summer School, in less than 10 days. My role was to create and maintain the tornado server, while other team members were working on the database management, styling, template engine etc.

Wish List was created in Python, HTML5, CSS3 and SQLite.
